## Introduction

A simple and straightforward gravity and orbit simulator to see how different bodies and masses affect each other. Just edit the python file to experiment with different bodies.

## Installation

```bash
> Windows
pip install pygame
python src/sim

> Mac/Linux
pip3 install pygame
python3 src/sim
```
